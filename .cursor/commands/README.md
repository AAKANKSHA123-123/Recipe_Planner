# Cursor Commands

Commands for the Smart Recipe & Meal Planner. Invoke via slash menu or natural language.

| Command | File | Description |
|---------|------|-------------|
| `/meal-plan` | [generate-meal-plan.md](generate-meal-plan.md) | Generate weekly meal plan from ingredients |
| `/export-shopping-list` | [export-shopping-list.md](export-shopping-list.md) | Export shopping list as CSV |

## Natural Language Triggers
- "Generate weekly meal plan" → generate-meal-plan
- "Export shopping list" → export-shopping-list
